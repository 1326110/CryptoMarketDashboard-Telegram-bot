# Telegram Bot Profile Setup

Instructions for configuring MarketPulse on Telegram's @BotFather. A complete, professional profile is required per Telegram's ad policies.

---

## Avatar

Upload a high-resolution image (minimum **512x512 px**, preferably 1024x1024) through @BotFather using `/setuserpic`.

**Recommended design:**
- Clean, minimal cryptocurrency/ chart icon
- Dark background with accent color (e.g., electric blue #0088FF or emerald #00E676)
- No text overlay
- No flashy or gambling-related imagery

---

## Bio

Set via @BotFather → `/setbio`.

**Suggested bio (max 512 characters):**

> Real-time cryptocurrency market dashboard. Track prices, monitor trends, and manage your portfolio — all inside Telegram. Data from CoinGecko.

**Alternative short bio:**  
> Cryptocurrency market monitoring tool. Price alerts, portfolio tracking, and market insights.

**Avoid in bio:**
- ❌ "Trade", "invest", "profit", "earn"
- ❌ Financial advice claims
- ❌ Emojis that suggest rockets, moons, or gambling (🚀, 🌙, 🎰)
- ❌ External links or contact info beyond @username

---

## Description

Set via @BotFather → `/setdescription`.

**Suggested description (max 512 characters):**

> MarketPulse helps you monitor cryptocurrency markets in real time. Use the dashboard to check prices, view the top 10 coins by market cap, set custom price alerts, and track your portfolio — all without leaving Telegram.
>
> Commands:
> /price <coin> — Check current price
> /top — Top coins by market cap
> /alert <coin> <target> — Set a price alert
> /portfolio — Manage your holdings
> /privacy — Data handling policy
> /terms — Terms of use

---

## Commands List

Set via @BotFather → `/setcommands`.

```
price - Check a coin's current price
top - Top 10 coins by market cap
alert - Set a price alert
portfolio - Manage your portfolio
privacy - Privacy policy
terms - Terms of use
help - Show available commands
```

---

## Profile Verification Checklist

- [ ] Avatar uploaded (high-resolution, professional design)
- [ ] Bio set (no profit language, matches bot function)
- [ ] Description set (lists features accurately)
- [ ] Commands registered in @BotFather
- [ ] No external links in bio
- [ ] No banned keywords in any profile field
- [ ] Bot username is clean and branded (no random strings)
