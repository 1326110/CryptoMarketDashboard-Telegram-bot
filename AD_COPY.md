# Ad Copy Samples

Compliant ad copy for MarketPulse, following Telegram's advertising policies ([`skill.md` Section 7](TG_Ads_Rule/skill.md)).

## Requirements Enforced

| Rule | Constraint |
|---|---|
| Max length | 160 characters |
| No imperative commands | Use curiosity / description, not "Click here" |
| No hype or exaggerated claims | Neutral, factual tone |
| Curiosity-based phrasing | Invite exploration, don't promise outcomes |
| Perfect grammar | Proofread before submission |
| No line breaks | Single paragraph |
| Max 1 Telegram internal link | Prefer `@MarketPulseBot` |
| Match ad promise with bot functionality | Ad must reflect actual features |
| No banned phrases | "Buy now", "Best", "#1", "Guaranteed", "Risk-free", "100x" |

---

## Pre-Submission Checklist

- [ ] Ad ≤ 160 characters
- [ ] No imperative commands ("Buy", "Click", "Start now")
- [ ] No earnings, profit, or return claims
- [ ] Single Telegram-native link (`@username` or `t.me/...`)
- [ ] Grammar and spelling verified
- [ ] Ad promise matches actual bot features
- [ ] Not reused from a previously banned campaign
